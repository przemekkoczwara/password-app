# Frontend Mentor - Password generator app

![Design preview for the Password generator app coding challenge](./src/assets/design/screenshot.pngjpg)

A simple solution for the [Frontend Mentor](https://www.frontendmentor.io) challenge - Password generator app

This project was created to practive HTML & CSS & RWD following the disign in Figma.

---

## Overview

This project recreates a **Article Preview Component** from a Figma design, including:

- Semantic HTML structure
- Responsive layout (mobile-first)
- Styled text hierarchy
- Optimized image display
- Clean CSS
- JavaScript - updateSliderStyle(),generatePassword(),
